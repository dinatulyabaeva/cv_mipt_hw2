# Обучение диффузионной модели на датасете котов

## Выбранный датасет

Использован датасет `huggan/few-shot-cat` с Hugging Face. Изображения приводились к размеру 64×64, переводились в RGB, нормализовались в диапазон [-1, 1].

## Архитектура модели

Использована модель `UNet2DModel` из библиотеки `diffusers`. Модель принимает зашумлённое изображение и номер шага диффузии, после чего предсказывает шум, который был добавлен к исходному изображению.

## Обучение

Обучение выполнялось по схеме DDPM. На каждом шаге к изображению добавлялся случайный гауссовский шум. Модель обучалась предсказывать этот шум. В качестве функции потерь использовалась MSE между настоящим шумом и предсказанным шумом.

## Параметры обучения

- Размер изображений: 64×64
- Batch size: 16
- Количество эпох: 30
- Количество diffusion timesteps: 1000
- Optimizer: AdamW
- Learning rate: 1e-4

## Генерация

Для генерации использовались два метода:

- DDPM: 1000 шагов
- DDIM: 100 шагов

## Сравнение DDPM и DDIM

DDPM использует больше шагов и обычно даёт более стабильную генерацию, но работает медленнее. DDIM позволяет пропускать часть шагов обратного процесса, поэтому генерация происходит быстрее.

Результаты замера времени:

- DDPM: 1000 шагов, 64.15 секунд
- DDIM: 100 шагов, 6.40 секунд

## Выводы

В ходе работы был реализован полный цикл обучения диффузионной модели: загрузка датасета, предобработка изображений, добавление шума, обучение модели предсказывать шум, сохранение чекпоинтов и генерация изображений. DDIM оказался быстрее DDPM, так как использует меньше шагов генерации.
